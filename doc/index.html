Ohmulator Documentation

Welcome to the documentation for Ohmulator, a hardware component emulator built in C++. This project aims to help users simulate the behavior of basic and complex electronic components.

About the Project

Ohmulator is an open-source project developed for educational purposes and fun. It simulates electronic components such as resistors, capacitors, inductors, transistors, and more. The project aims to help users learn about electronics in an interactive, code-driven environment.
