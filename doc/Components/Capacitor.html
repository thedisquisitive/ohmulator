Capacitor

A capacitor is a passive component that stores energy in an electric field.

Overview

Capacitors are used to smooth voltage fluctuations, store energy, and filter signals in circuits. The capacitance determines the amount of electric charge stored for a given voltage.

Parameters

  • Capacitance: The value in Farads, e.g., 1000µF.
  • Voltage Rating: The maximum voltage the capacitor can handle.

Methods

  • applyVoltages(double inputVoltage, double groundVoltage)
  • output() const
  • visualize() const
  • getName() const

Example


// Create a 1000µF capacitor
Capacitor smoothingCap(1000);
circuit.addComponent(&smoothingCap);
std::cout << "Output: " << smoothingCap.output() << std::endl;
