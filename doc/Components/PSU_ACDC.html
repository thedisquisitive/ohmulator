AC to DC Power Supply Unit (PSU) Component

The PSU_ACDC class simulates an AC to DC power supply unit, converting AC voltage to stable DC voltage using a rectifier and smoothing capacitor.

Class Definition


class PSU_ACDC {
public:
    PSU_ACDC(double acAmplitude, double acFrequency);

    void turnOn();
    void turnOff();
    double getDCOutput(double time) const;  // Get DC output at a given time

private:
    ACPowerSource acSource;
    Rectifier rectifier;
    Capacitor smoothingCap;
    Switch powerSwitch;
    bool isOn;
};

Constructor

  • PSU_ACDC: Initializes the PSU with an AC power source, rectifier, and smoothing capacitor.

Methods

  • turnOn: Turns on the PSU, allowing it to convert AC to DC.
  • turnOff: Turns off the PSU.
  • getDCOutput(double time): Returns the DC output at a given time, after rectification and smoothing.

Usage Example


PSU_ACDC psu(230, 50);  // 230V AC, 50Hz
psu.turnOn();
double dcOutput = psu.getDCOutput(0.02);  // Get DC output at time 0.02s

Behavior

The PSU takes an AC input, converts it to DC using a rectifier, and smooths the output using a capacitor. It is typically used in circuits where stable DC voltage is required from an AC power source.
