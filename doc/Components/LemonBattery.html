Lemon Battery

A Lemon Battery generates electrical energy from the chemical reaction between the lemon's acid and two electrodes.

Overview

Lemon batteries are a common educational project to demonstrate how chemical reactions can generate electricity. They use the citric acid in lemons to create a small voltage.

Methods

  • getName() const - Returns the name of the battery.

Example


// Create a Lemon Battery
LemonBattery batt;
std::cout << "Battery: " << batt.getName() << std::endl;
