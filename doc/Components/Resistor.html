Resistor

A resistor limits the current flowing through a circuit by providing resistance.

Overview

Resistors are passive components that reduce current flow, divide voltages, and dissipate energy in a circuit. The resistance value determines how much the current is reduced.

Parameters

  • Resistance: The value in ohms (Ω).

Methods

  • applyVoltages(double inputVoltage, double groundVoltage)
  • output() const
  • visualize() const
  • getName() const

Example


// Create a 1kΩ resistor
Resistor r(1000);
circuit.addComponent(&r);
std::cout << "Output: " << r.output() << std::endl;
