Transistor Component

The Transistor class simulates both N-type and P-type transistors, which are used to control current flow based on the input voltages at the gate, source, and drain terminals.

Class Definition


class Transistor : public Component {
public:
    enum class Type { NType, PType };

    Transistor(Type type, double thresholdVoltage);
    void applyVoltages(double inputVoltage, double groundVoltage = 0) override;
    double output() const override;
    std::string visualize() const override;
    std::string getName() const override;

private:
    Type type;
    double thresholdVoltage;
    bool conducting;
    double gateVoltage;
    double sourceVoltage;
    double drainVoltage;
};

Methods

  • applyVoltages: Simulates the transistor behavior based on input voltages.
  • output: Returns the current through the transistor.
  • visualize: Provides a string representation of whether the transistor is conducting.
