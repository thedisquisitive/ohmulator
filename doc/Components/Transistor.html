Transistor

A transistor is a semiconductor device used to amplify or switch electronic signals.

Overview

Transistors are key components in electronics, used to amplify signals or act as electronic switches. They can be either N-type or P-type and are controlled by the voltage at the gate.

Parameters

  • Type: N-type or P-type.
  • Threshold Voltage: The voltage needed to switch the transistor on.

Methods

  • applyVoltages(double inputVoltage, double groundVoltage)
  • output() const
  • visualize() const
  • getName() const

Example


// Create an N-type transistor with a 0.7V threshold
Transistor t(Transistor::Type::NType, 0.7);
circuit.addComponent(&t);
std::cout << "Output: " << t.output() << std::endl;
