Alkaline Battery

An Alkaline Battery is a common type of dry cell battery that provides stable voltage output over time.

Overview

Alkaline batteries are widely used in portable electronic devices. They offer stable voltage output for extended periods, making them reliable power sources for a variety of applications.

Methods

  • getName() const - Returns the name of the battery.

Example


// Create an Alkaline Battery
AlkalineBattery batt;
std::cout << "Battery: " << batt.getName() << std::endl;
