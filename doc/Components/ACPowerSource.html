AC Power Source Component

The ACPowerSource class simulates an alternating current (AC) power source, providing a sinusoidal voltage over time.

Class Definition


class ACPowerSource : public Component {
public:
    ACPowerSource(double amplitude, double frequency);

    void applyVoltages(double inputVoltage, double groundVoltage = 0) override;
    double output() const override;
    double output(double time) const;  // Time-varying AC voltage
    std::string visualize() const override;
    std::string getName() const override;

private:
    double amplitude;  // Peak voltage
    double frequency;  // Frequency of the AC signal (Hz)
    mutable double currentOutput;  // Store the current output voltage
};

Constructor

  • ACPowerSource: Initializes the AC power source with a given peak voltage and frequency.

Methods

  • output(double time): Returns the AC voltage at a given time, simulating a sinusoidal waveform.
  • output(): Returns the most recent output voltage, updated by the time-based output method.
  • visualize(): Provides a string representation of the AC source.

Usage Example


ACPowerSource acSource(230, 50);  // 230V peak, 50Hz frequency
double acVoltage = acSource.output(0.02);  // Get voltage at time 0.02s

Behavior

The AC power source generates a sinusoidal voltage over time. It is typically used in circuits requiring an alternating current, such as rectifiers for converting AC to DC.
