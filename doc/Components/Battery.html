Battery

A battery provides a constant voltage and stores energy to power circuits.

Overview

Batteries are key power sources in circuits, providing a steady DC voltage. Batteries in Ohmulator have parameters like voltage and capacity (in mAh).

Parameters

  • Voltage: The constant voltage provided by the battery.
  • Capacity: The amount of energy stored, measured in milliamp hours (mAh).

Methods

  • applyVoltages(double loadVoltage, double groundVoltage)
  • output() const
  • discharge(double loadCurrent)
  • visualize() const
  • getName() const

Example


// Create a 5V battery with 2000mAh capacity
Battery batt(5.0, 2000);
circuit.addComponent(&batt);
std::cout << "Output: " << batt.output() << std::endl;
