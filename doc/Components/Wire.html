Wire

A wire transmits electrical signals and connects components in a circuit.

Overview

Wires are used to connect different components in a circuit and transmit current between them. They play a fundamental role in ensuring a functional circuit.

Methods

  • connect(bool isConnected)
  • applyVoltages(double inputVoltage, double groundVoltage)
  • output() const
  • visualize() const
  • getName() const

Example


// Create a wire and connect it
Wire wire;
wire.connect(true);
circuit.addComponent(&wire);
std::cout << "Output: " << wire.output() << std::endl;
