LED

An LED emits light when a current flows through it in the forward direction.

Overview

LEDs are commonly used in circuits as indicators or for lighting. They allow current to pass in one direction and emit light when forward-biased.

Parameters

  • Forward Voltage: The voltage needed for the LED to emit light.

Methods

  • applyVoltages(double inputVoltage, double groundVoltage)
  • output() const
  • visualize() const
  • getName() const

Example


// Create an LED with a forward voltage of 1.8V
LED led(1.8);
circuit.addComponent(&led);
std::cout << "Output: " << led.output() << std::endl;
