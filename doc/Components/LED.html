LED Component

The LED class simulates a light-emitting diode. It emits light when forward biased and the input voltage exceeds its forward voltage.

Class Definition


class LED : public Component {
public:
    LED(double forwardVoltage);
    void applyVoltages(double inputVoltage, double groundVoltage = 0) override;
    double output() const override;
    std::string visualize() const override;
    std::string getName() const override;

private:
    double forwardVoltage;
    bool lit;
    double current;
};

Methods

  • applyVoltages: Simulates the LED behavior, turning it on if the forward voltage is exceeded.
  • output: Returns the current through the LED.
  • visualize: Provides a string representation of whether the LED is lit.
