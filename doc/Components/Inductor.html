Inductor

An inductor stores energy in a magnetic field when electric current flows through it.

Overview

Inductors are used to oppose changes in current and are key in building filters, transformers, and chokes. The inductance value is measured in Henrys (H).

Parameters

  • Inductance: The property that resists changes in current.

Methods

  • applyVoltages(double inputVoltage, double groundVoltage)
  • output() const
  • visualize() const
  • getName() const

Example


// Create a 5H inductor
Inductor coil(5);
circuit.addComponent(&coil);
std::cout << "Output: " << coil.output() << std::endl;
