MagneticField Component

The MagneticField class simulates a static magnetic field, such as the one produced by a magnet in a guitar pickup.

Class Definition


class MagneticField {
public:
    MagneticField(double strength);
    
    // Simulate the field strength based on distance from the string
    double calculateFieldStrength(double distance) const;

private:
    double strength;  // Base strength of the magnetic field
};

Methods

  • calculateFieldStrength: Calculates the magnetic field strength at a specific distance from the source.
