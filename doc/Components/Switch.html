Switch

A switch controls the flow of current in a circuit by opening or closing the circuit path.

Overview

Switches are used to control whether or not current flows through a circuit. The state of the switch (on/off) determines the connectivity of the circuit.

Parameters

  • Initial State: The starting state of the switch (on/off).

Methods

  • setState(bool state)
  • applyVoltages(double inputVoltage, double groundVoltage)
  • output() const
  • visualize() const
  • getName() const

Example


// Create a switch that starts in the off position
Switch sw(false);
circuit.addComponent(&sw);
sw.setState(true); // Turn on the switch
std::cout << "Output: " << sw.output() << std::endl;
